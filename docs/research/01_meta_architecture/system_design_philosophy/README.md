# System Design & Philosophy

Research on foundational design principles for autonomous AI coding systems, including KISS/modularity constraints, anti-slop architecture patterns, complexity budgets, spec-driven vs intent-driven development paradigms, and architecture drift detection mechanisms.

## Core Artifacts

- [overview.md](./overview.md) - Executive summary, topic framing, and detailed synthesis
- [comparisons.md](./comparisons.md) - Comparative tables for design approaches and tools
- [patterns.md](./patterns.md) - Design patterns and anti-patterns
- [references.md](./references.md) - Full source bibliography with metadata

## Key Research Questions

1. How do BDI architectures complement LLM-based agents for accountable autonomy?
2. What complexity scoring frameworks prevent scope creep in agentic SDLC?
3. How do spec-driven workflows compare to intent-driven approaches?
4. What mechanisms detect and prevent architecture drift in evolving codebases?

## Related Topics

- **Economic & Optimization Modeling**: Token budgets and cost constraints
- **Governance & Compliance**: Audit trails for design decisions
- **Security Architecture**: Sandboxing and privilege isolation
