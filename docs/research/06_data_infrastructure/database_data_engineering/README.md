# Database & Data Engineering

## Overview

This topic covers the principles, patterns, and practices for managing data infrastructure in autonomous AI coding systems. It encompasses schema management, migration control, data validation contracts, data drift detection, synthetic data generation, test data lifecycle management, database management, API management, and API interaction mapping.

Database and data engineering practices ensure data integrity, schema consistency, and reliable data pipelines—critical foundations for autonomous AI coding systems that must reason about, generate, and evolve database schemas and data flows as part of software development workflows.
