- **Limitations/biases**: Network slicing specific.

- **Limitations/biases**: 6G networking specific.

**[kimi:9] Haroon & Schuler (2025)** Distributed Area Coverage with High Altitude Balloons Using Multi-Agent Reinforcement Learning. Type: paper. https://arxiv.org/abs/2510.03823
- **Main contribution**: First systematic MARL application to HAB coordination. QMIX with Centralized Training Decentralized Execution for atmospheric vehicle coordination.
- **Limitations/biases**: Aerospace domain specific.

**[kimi:10] Masters et al. (2025)** Orchestrating Human-AI Teams: The Manager Agent as a Unifying Research Challenge. Type: paper. https://arxiv.org/abs/2510.02557

- **Limitations/biases**: Research vision paper.
