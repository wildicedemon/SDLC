# Specification & Design

## Description

Specification & Design encompasses the practices, templates, and standards that shape how autonomous AI coding agents create, maintain, and align code with intended behavior. This topic covers documentation templates, architecture documentation, API flow diagrams, database schema management, coding style guidelines, complexity management frameworks, test-driven/behavior-driven/intent-driven development approaches, design standards, and critiques of spec-driven development paradigms.

## Relevance to Autonomous AI Coding

Specification & Design is critical to agentic SDLC as it:
1. Provides templates that ensure consistent, maintainable code generation
2. Establishes guardrails against scope creep and complexity explosion
3. Enables verification that generated code matches intended behavior
4. Creates documentation that supports future agent sessions
5. Balances specification rigor with implementation flexibility
